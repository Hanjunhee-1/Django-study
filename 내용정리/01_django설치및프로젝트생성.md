## ※ 당연히 python 이 설치되어 있어야 django 를 사용할 수 있습니다!
<br/><br/>


# Django 설치하기
```bash
    python -m pip install Django
```
<br/><br/><br/><br/>


# Django 버전 확인하기
```bash
    python -m django --version
```
<br/><br/><br/><br/>


# Django 프로젝트 생성하기
```bash
    django-admin startproject [project_name]
```
※ 프로젝트 이름을 정할 때 Django 에서 사용중인 이름은 피해야 합니다. <br/>
ex) django, test, ...
